# 项目记忆

这个目录用于保存仓库级的跨会话上下文，帮助后续任务更快建立项目理解。

## 阅读顺序

1. `generated/project-summary.md`
2. `generated/development-playbook.md`
3. `generated/feature-index.md`
4. `memory/current.md`
5. `memory/decisions.md`
6. `memory/milestones.md`

## 规则

- `generated/` 可根据仓库现状由 AI 直接刷新。
- `memory/` 存放人工维护或长期保留的信息，不应被自动覆盖。
- 除非用户明确要求其他语言，新增或刷新内容默认使用中文。
